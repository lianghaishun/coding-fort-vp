# 技术栈规范

## TypeScript 与 JavaScript

### TypeScript 优点

- 静态类型语言要较于动态类型语言更安全；
- TypeScript 代码更可靠且更易于重构；
- TypeScript 更明确类型；

## Sass 与 Less

### Sass 优点

- Sass 功能齐全；
- 不仅有变量和作用域，还有函数和进程控制的概念；
- 更像一门正规的编程语言；

## Vite 与 Webpack

### Vite 优点

- 开发效率极高 —— 🚀 般的速度；
- 上手极其简单；
- 社区成本低；
- 目前业务内所有项目已慢慢转为 Vite 构建；
